# Gestión del proyecto

## Metodología

Para organizar el desarrollo de GameCrew estoy usando un tablero Kanban en Trello.
He elegido Kanban porque al trabajar sola puedo mover las tareas libremente según
voy avanzando, sin necesidad de sprints fijos.

## Estructura del tablero

El tablero tiene cinco columnas:

- **Backlog**: funcionalidades definidas pero no empezadas
- **Todo**: tareas listas para empezar
- **In Progress**: en lo que estoy trabajando ahora
- **Review**: terminado pero pendiente de revisar
- **Done**: completado

## Tarjetas

Cada funcionalidad principal de la app tiene su propia tarjeta con una checklist
de subtareas técnicas. Esto me permite ver el progreso dentro de cada funcionalidad
y no perder el hilo de lo que falta.
